Dato in input il file text.txt che contiene le parole del testo dell'antico giuramento di Ippocrate separate da spazi (il file contiene solo lettere e spazi).
Realizzare un programma che scriva sul file words.txt la lista di parole contenute nel testo con il numero di occorrenze (quante volte una parola appare nel testo). Il file dovrà contenere, ordinate in ordine alfabetico crescente, una per ogni riga, le parole seguite dal numero di occorrenze separato da una virgola (ovviamente ogni parola dovrà essere riportata una sola volta, seguita dal numero totale di occorrenze di quella parola). 
Le prime righe del file words.txt dovrebbero essere uguali alle seguenti:
a,5
abortivo,1
accada,1
ad,1
adempio,1
alcuno,1
allievi,1
altra,1
altro,2
anche,1
andrò,1
apollo,1
...

BONUS
Scrivere in un nuovo file "words_bonus.txt", con lo stesso formato del precedente, le parole ordinate per numero di occorrenze in ordine decrescente (dalla più comune alla meno comune).
Le prime righe del file words_bonus.txt dovrebbero essere uguali alle seguenti:
e,29
di,9
il,8
che,7
mio,6
per,6
un,6
a,5
se,5
arte,4
degli,4
io,4
...
